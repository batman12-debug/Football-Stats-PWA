# GoalMind API

FastAPI backend for the GoalMind World Cup 2026 app.

- Health: `GET /health`
- Teams: `GET /api/teams`
- Matches: `GET /api/matches/upcoming`

Deployed from [Football-Stats-PWA](https://github.com/batman12-debug/Football-Stats-PWA).
